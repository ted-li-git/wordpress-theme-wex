## 📌 主题简介

Wex 是一款轻量级、现代化的 WordPress 主题，专为应用展示、作品展示等内容型网站设计。支持区块编辑、响应式布局，易于定制。

---

## ✨ 主题特色

- ✅ 支持 WordPress 全站编辑
- ✅ 响应式布局，适配手机、平板、桌面
- ✅ 内置多种区块样式和模板
- ✅ 支持特色图像和文章格式
- ✅ 兼容插件

---

## 🚀 安装方法

1. 下载主题压缩包（ZIP 文件）；
2. 登录 WordPress 后台 → 外观 → 主题 → 添加新主题 → 上传主题；
3. 上传 ZIP 文件并点击“立即安装”；
4. 安装完成后点击“启用”。

---

## 🛠️ 自定义开发

如需对主题进行自定义：

- 打开区块编辑器，直接编辑即可！
